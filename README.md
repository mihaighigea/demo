Clone the app:
```$bash
$ git clone https://github.com/mihaighigea/demo.git 
```

Run the app:
```$bash
$ mvn spring-boot:run
```

URL for accessing the H2 console:
```$bash
http://localhost:8080/h2-console/
```

URL for accessing the Swagger documentation:
```$bash
http://localhost:8080/swagger-ui.html
```
